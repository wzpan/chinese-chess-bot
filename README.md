# chinese-chess-bot

中国象棋频道机器人，通过纯文本的方式进行下棋。

当机器人拥有管理员权限的时候，如果有成员赢了机器人，将自动创建 “象棋大师” 身份组并为该成员添加 “象棋大师” 荣誉称号。

## 使用方法

```
功能菜单：

/开局
    开一局游戏
/下棋 行1列1行2列2
    根据步法下棋
    示例： /下棋 h2e2
/投降
    投降认输。只有开局的人才能投降
```
